FTP :
 
1) Make Connection
2) Assign IP to server
3) Assign IP to PC
   Default Gateway is same as IP of server
4) services -> Turn Every Service other than FTP off
5) create user, password and set permission for user.
6) come to any PC
   command prompt
   ipconfig
   ftp ip_address
   Enter username and password
   when set to FTP
   dir
   delete File_name
